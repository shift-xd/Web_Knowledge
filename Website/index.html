DEFINITION OF INVERTIBLE FUNCTION In mathematics, a function is called invertible if there exists another function that "undoes" the operation of the original function. The inverse function is written as f⁻¹(x). A function f(x) is invertible if and only if: f⁻¹(f(x)) = x and f(f⁻¹(x)) = x for all values of ‘x’ in the domain of the original and inverse functions. This means that the inverse function essentially reverses the effect of the original function. For instance, if f takes a to b, then f⁻¹ takes b back to a. HORIZONTAL LINE TEST The Horizontal Line Test is a visual method to determine whether a function is invertible. A function is invertible only if it is one-to-one – meaning no two inputs give the same output. How the Test Works: If every horizontal line intersects the graph at most once, the function is one-to-one and invertible. If any horizontal line cuts it more than once, the function is not invertible unless we restrict its domain. For example: f(x) = x + 2 → Passes the test while f(x) = x² → is a failure. CONDITIONS FOR INVERTIBLE FUNCTION Not every function has an inverse. For a function to be invertible, it must satisfy the following conditions: One-to-One Function (Injective): A function is one-to-one if different inputs give different outputs. That is, for a function f(x): If f(a) = f(b), then a = b If two inputs produce the same output, the function is not one-to-one and cannot have an inverse. Example: f(x) = x² is not one-to-one over all real numbers since: f(2) = 4 and f(-2) = 4.
